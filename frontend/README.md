# Sistema de Parcerias para Corretora de Seguros

Este é um sistema completo para gestão de parceiros que indicam clientes para uma corretora de seguros.

## Funcionalidades

- Cadastro e login de parceiros
- Indicações de clientes
- Comissões de 50%
- Central de aprendizado
- Gamificação e ranking

Veja as instruções na pasta `/docs`
